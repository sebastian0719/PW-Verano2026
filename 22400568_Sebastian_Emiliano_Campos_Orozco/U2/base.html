Propiedades fundamentalesPara que un mensaje sea considerado un texto, debe cumplir con tres características principales:Coherencia: Las ideas deben estar organizadas de forma lógica para que el tema central se entienda.Cohesión: Las oraciones y párrafos deben estar conectados entre sí utilizando los conectores y la gramática adecuada.Adecuación: El mensaje debe ser apropiado para el contexto, el emisor y el receptor.Tipos de texto más comunesNarrativo: Cuenta una historia o evento (ej. novelas, cuentos).Expositivo: Explica o transmite información de manera objetiva (ej. artículos académicos, enciclopedias).
